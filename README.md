# WorldEdit Converter

> Fork of [multi-nono/3D-Model-To-world.edit-command-minetest-W.E-](https://github.com/multi-nono/3D-Model-To-world.edit-command-minetest-W.E-)

Convert Minecraft Schematics and 3D models to Multicraft WorldEdit commands.

## Features

- `.schem`, `.nbt`, `.litematic` → WorldEdit commands
- Minetest WorldEdit `.we` files → WorldEdit commands
- 3D models (`.obj`, `.stl`) → Voxelized commands
- Browser-based (no installation required)

## Usage

Open `index.html` in a browser, drop a file, and click "Convert".

## Directory Structure

```
├── index.html / index_en.html  # Web converter (JP/EN)
├── assets/                     # Images
├── js/                         # JavaScript modules
└── scripts/                    # Python CLI tools
```
